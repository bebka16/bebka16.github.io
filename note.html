Возможности Snail Note

Snail Note создан для тех, кто ценит скорость и не любит перегруженные интерфейсы. Редактор не пытается быть средой разработки, но даёт всё необходимое для комфортной работы с кодом на разных языках программирования.

  • Подсветка синтаксиса для десятка языков программирования
  • Магазин расширений и тем, которые пополняются сообществом
  • Система вкладок и разделение окна
  • Возможность предпросмотра HTML
  • Удобный проводник файлов для больших проектов
  • Автосохранение и восстановление сессий
