Snail Note

Выделенный редактор кода без отвлекающих функций. Минималистичный интерфейс, быстрая работа с большими файлами и интеллектуальная подсветка синтаксиса для повседневной разработки.

Возможности Snail Note

Snail Note создан для тех, кто ценит скорость и отсутствие визуального шума. Редактор не пытается быть средой разработки, но даёт всё необходимое для комфортной работы с кодом и текстом.

  • Подсветка синтаксиса для 40+ языков программирования
  • Мгновенное открытие файлов размером до 100 МБ
  • Система вкладок и разделение окна
  • Встроенная тёмная и светлая темы
  • Поддержка регулярных выражений в поиске
  • Автосохранение и восстановление сессий
