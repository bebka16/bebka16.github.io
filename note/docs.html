Введение

Snail Note — лёгкий, но мощный редактор кода, вдохновлённый современными IDE. Он построен на базе Monaco Editor (движок VS Code) и работает как настольное приложение (Electron) или в браузере. Основные возможности: подсветка синтаксиса для десятков языков, вкладки, проводник файлов, поиск с регулярными выражениями, поддержка пользовательских тем и расширений.

Интерфейс

Верхняя панель

  • Файл — создание, открытие, сохранение, закрытие вкладок.
  • Правка — поиск, замена, комментирование, форматирование.
  • Вид — показать/скрыть проводник, масштаб интерфейса.
  • Язык — переключение языка интерфейса (RU/BY/UA/EN).
  • Настройки — общие, язык, уведомления, история, о программе.
  • Расширения — установка и управление расширениями и темами.
  • Помощь — ссылка на эту документацию.

Строка состояния

  • Позиция курсора (строка:столбец)
  • Размер выделения (строк × символов)
  • Путь к текущему файлу
  • Кодировка (UTF-8)
  • Язык программирования (выпадающий список)
  • Символы конца строки (LF / CRLF)

i Статус-бар можно отключить в настройках.

Работа с файлами и папками

  • Новый файл — создаёт вкладку untitled_*.txt.
  • Открыть файл — диалог выбора файла. Поддерживаются изображения (PNG, JPG, GIF, SVG и др.) — они открываются во встроенном просмотрщике.
  • Открыть папку — показывает содержимое папки в проводнике слева. Можно навигировать по подпапкам, кликать по файлам для открытия.
  • Сохранить — сохраняет текущий файл. Если файл безымянный — откроется диалог «Сохранить как».
  • Сохранить как — сохранить копию под новым именем.
  • Закрыть вкладку — закрывает текущий файл (без подтверждения, но история сохраняется).
  • Кнопка ✕ (закрыть все) на панели быстрых действий закрывает все вкладки.

Проводник — показывает дерево файлов. Поддерживается навигация «Назад» и кнопка очистки. Размер проводника можно менять перетаскиванием правой границы.

Редактирование

  • Автоматическая подсветка синтаксиса для JavaScript, TypeScript, HTML, CSS, Python, PHP, Java, C++, Go, Ruby, Rust, SQL, XML, YAML, Markdown, Shell и других.
  • Язык можно переключить вручную через выпадающий список в статус-баре или через меню «Язык».
  • Закомментировать — добавляет/убирает комментарий для текущей строки или выделенного блока (Ctrl+/).
  • Форматировать — автоматически выравнивает код (доступно для JS, TS, HTML, CSS, JSON, Markdown).
  • Отменить / Повторить — стандартные Ctrl+Z / Ctrl+Y.
  • Контекстное меню (ПКМ) — поиск, замена, комментарий, форматирование, копировать/вырезать/вставить, поиск в Google, смена регистра.

Поиск и замена

  • Найти — Ctrl+F (открывается панель поиска).
  • Заменить — Ctrl+H (появляется поле замены).
  • Опции: учитывать регистр, целое слово, регулярные выражения (синтаксис PCRE).
  • Навигация по совпадениям: кнопки «Найти» (следующее), «Назад» (предыдущее).
  • Кнопки «Заменить» (одно) и «Все» (заменить все).
  • Счётчик совпадений отображается справа от опций.

Настройки и темы

  • Общее: включение/выключение статус-бара, иконок языков на вкладках.
  • Язык интерфейса: русский, белорусский, украинский, английский (выбор сохраняется).
  • Уведомления об обновлениях — при наличии новой версии показывается диалог.
  • История — список открытых/сохранённых файлов с возможностью повторного открытия или удаления записи.
  • О программе — версия, автор, ссылки.
  • Масштаб — отдельно размер шрифта редактора (10–30px) и общий масштаб интерфейса (80%–150%).

Расширения и темы оформления

  • Snail Note поддерживает загрузку пользовательских расширений (JavaScript) и тем (CSS).
  • Через меню «Расширения» → «Расширения» открывается страница управления (встроенный iframe с каталогом).
  • «Расширения» → «Темы» — позволяет выбрать и применить готовую тему Monaco.
  • Темы можно устанавливать через custom-theme-link — автоматически парсится CSS-правило с объявлением theme: "vs-dark" или --theme: "hc-black".
  • Расширения, загруженные как Blob-URL, кешируются и восстанавливаются при перезапуске.
